#Repositorio de Aprendizaje de Programacion Declarativa

Cada uno de los archivos en este repositorio reflejan un aspecto a aprender sobre la programacion declarativa.
Desde el inicio de este curso, y las bases de prolog hasta las operaciones matematicas y el uso de arboles binarios, este repositorio recopila tanto los ejercicios por cada tema como su explicacion para un mayor entendimiento.
